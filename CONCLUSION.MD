# Project Conclusion

The Survey Probability Analysis project was developed using Google Sheets and Python.

The analysis showed that Instagram was the most preferred social media platform among respondents, while Facebook had the lowest preference.

The project included:
- Frequency Distribution
- Probability Distribution
- Percentage Analysis
- Conditional Probability
- Expected Value Prediction
- Random Simulation
- Dashboard Visualization

The results demonstrate how probability concepts can be applied to real-world survey data.
